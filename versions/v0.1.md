# v0.1 — Hand & Body Segmentation

## Release Date
2026-05-12

## Summary
First pipeline stage: detect and segment the egocentric person's hands and body
using SAM3 (text prompt) and SAM2 (point prompt), producing two visualization videos.

## Features Added
- SAM3 text-prompt hand detection ("egocentric first person's hands or gloves")
- Left/right hand assignment via centroid-x heuristic with temporal consistency
- SAM2 body segmentation using point prompt at lower-center of frame
- Output video 1: hand masks with colored overlays and L/R labels
- Output video 2: body mask with dilated hand masks subtracted
- Metadata JSON export with frame counts and settings

## How to Run
```bash
conda activate sam3
python src/segment.py data/input.mp4

# Process only first 10 seconds:
python src/segment.py data/input.mp4 --seconds 10

# Custom output directory:
python src/segment.py data/input.mp4 -o outputs/
```

## Dependencies
- Python 3.12 (sam3 conda environment)
- SAM3 (editable install from hand_pose_4090/sam3/)
- SAM2 (sam2.1_hiera_large checkpoint)
- OpenCV, NumPy, SciPy, PyTorch

## Notes
- Uses `conda activate sam3` environment which has both SAM2 and SAM3
- SAM3 max_objects=2 to avoid detecting bystander hands
- Body point prompt at (W/2, H*0.85) — assumes ego person at bottom-center
- Hand dilation of 30px when subtracting from body mask
