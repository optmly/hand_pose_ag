# v0.0 — Project Scaffold

## Release Date
2026-05-12

## Summary
Initial project setup with versioned workflow tooling, goal-driven development
system, and auto-commit infrastructure.

## Features Added
- `.gitignore` excluding data, model weights, and caches
- `README.md` with architecture overview and quick start
- `Makefile` for common operations
- Version manifest system (`versions/`)
- Auto-commit script (`scripts/commit_and_push.sh`)
- Claude goal specification system (`goals/current_goals.yaml`)
- Pipeline configuration (`configs/pipeline.yaml`)
- Environment setup script (`scripts/setup_env.sh`)

## How to Run
```bash
# Set up the environment
bash scripts/setup_env.sh

# View available commands
make help

# View current goals for Claude
make goals
```

## Dependencies
- Python 3.11+
- PyYAML (for goal/config parsing)

## Notes
- No pipeline code yet — this version is scaffold only
- The `data/` folder contains `input.mp4` but is git-ignored
